gulp相关知识

[gulp](https //gulpjs.com/)是一个基于流的自动化构建工具，不包括模块化的功能，通过配置一系列的task，例如文件压缩合并、雪碧图、启动server、版本控制等，然后定义执行顺序来让gulp执行task，从而构建前端项目的流程。

开始

这里简单记录一下gulp的一些简单的概念。

核心API。

  • task 创建一个任务
  • series 顺序执行多个任务
  • prallel 并行执行多个任务
  • src 读取数据源转换成stream
  • pipe 管道-可以在中间对数据流进行处理
  • dest 输出数据流到目标路径
  • on 事件监听
  • watch 数据源监听

使用

  • 安装 npm i gulp -D

  • 创建gulpfile.js

  • 执行命令
    gulp copy

gulp会把demo.txt文件拷贝到dist/asset下面

插件

  • gulp-concat 文件合并
  • gulp-uglify js文件压缩
  • gulp-rename 文件重命名
  • gulp-less less文件编译
  • gulp-clean-css css压缩
  • gulp-livereload 实时自动编译刷新
  • gulp-plumber 防止中间任务出错而导致退出的
  • gulp-if 根据条件来判断是否执行指定任务
  • gulp-replace 文件内容替换
